# hatebu-watch

[![Slack Room][slack-badge]][slack-link]

Watch the Hatena Bookmark count of specific post

## Install

With [fisherman]

```
fisher tsub/hatebu-watch
```

## Usage

```fish
hatebu-watch <URL>
```
